# 🤖 Luky Messenger Userbot

بوت تلقائي لحساب فيسبوك ماسنجر شخصي، مبني باستخدام لغة **Node.js** ومكتبة **fca-project-orion**. يقوم البوت بالرد التلقائي على الرسائل الواردة ويحتوي على صلاحيات خاصة بالمطور (الآدمن).

---

## 👑 المطور
* **الاسم:** Fares Lkouchi
* **اسم البوت:** Luky

---

## 📂 مكونات المشروع الأساسية
* `index.js`: الكود الرئيسي لتشغيل البوت والاستماع للرسائل.
* `config.json`: ملف الإعدادات الخاص بمعرف الآدمن (Admin ID) وأسماء البوت والمطور.
* `package.json`: ملف إدارة الحزم والمكتبات المطلوبة للمشروع.
* `appstate.json`: ملف يحتوي على الكوكيز الخاصة بالحساب لتسجيل الدخول بأمان.

---
